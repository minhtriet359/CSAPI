API Documentation

Welcome to the API documentation for our service.

/encrypt-symmetric

POST Encrypt data using a symmetric key.


        {
            "data": "string",
            "key": "string"
        }

Response:


        {
            "encrypted_data": "string"
        }

/decrypt-symmetric

POST Decrypt data using a symmetric key.


        {
            "data": "string",
            "key": "string"
        }

Response:


        {
            "decrypted_data": "string"
        }

/encrypt-asymmetric

POST Encrypt data using an asymmetric key.


        {
            "data": "string",
            "key": "string"
        }

Response:


        {
            "encrypted_data": "string"
        }

/decrypt-asymmetric

POST Decrypt data using an asymmetric key.


        {
            "data": "string",

        }

Response:


        {
            "decrypted_data": "string"
        }

/hash

POST Compute hash of a string.


        {
            "data": "string"
        }

Response:


        {
            "hash_data": "string"
        }

/verify-hash

POST Verify hash of a string.


        {
            "data": "string",
            "hash": "string"
        }

Response:


        {
            "message": "Hash verification successful"
        }

Error Response:


        {
            "error": "Hash verification failed"
        }

/sign

POST Sign data using a key.


        {
            "data": "string",
            "key": "string"
        }

Response:


        {
            "signature": "string"
        }

/verify-signature

POST Verify data signature using a key.


        {
            "data": "string",
            "signature": "string",
            "key": "string"
        }

Response:


        {
            "message": "Signature verified"
        }

Error Response:


        {
            "error": "Signature and key do not match"
        }

/key/generate

POST Generate keys (symmetric or asymmetric).


        {
            "type": "symmetric" | "asymmetric",
            "algorithm": "AES" | "RSA"
        }

Response:


        {
            "key": "string"  // For symmetric keys
        }
    

        {
            "private_key": "string",
            "public_key": "string"  // For asymmetric keys
        }

/key/store

POST Store keys in the database.


        {
            "type": "symmetric" | "asymmetric",
            "key": "string" | {"private_key": "string", "public_key": "string"}
        }

Response:


        {
            "message": "Keys stored successfully"
        }

Error Response:


        {
            "error": "Failed to store keys"
        }

/key/get

GET Retrieve keys from the database.

Query Parameters:


        type=symmetric | asymmetric

Response:


        {
            "key": "string"  // For symmetric keys
        }
    

        {
            "private_key": "string",
            "public_key": "string"  // For asymmetric keys
        }

Error Response:


        {
            "error": "Failed to retrieve keys"
        }

/user/register

POST Register a new user.


        {
            "username": "string",
            "password": "string"
        }

Response:


        {

        }

Error Response:

        }

/user/login

POST Authenticate user login.


        {
            "username": "string",
            "password": "string"
        }

Response:


        {
            "access_token": "string"
        }

Error Response:


        {
            "error": "Invalid username or password"
        }
